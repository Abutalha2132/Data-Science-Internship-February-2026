## Project Description

This repository contains solutions to basic Python programming problems designed to strengthen core programming concepts such as lists, dictionaries, sets, tuples, loops, and conditional statements.

The problems are beginner-friendly and focus on logical thinking and real-world applications.

* Strings and Text Processing
* Lists and Iteration

* Sets (Removing Duplicates)
* Tuples (Immutable Data)
* Conditional Statements
* Counting and Searching Algorithms
